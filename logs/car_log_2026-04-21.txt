WHAT CAR DOES
-------------
1. Computes mean signal across all electrodes at each time point
2. Subtracts this "common average" from each channel
3. Removes global noise (movement, distant activity, reference bias)
4. Preserves local/focal activity (important for SOZ localization)

IMPLEMENTATION DETAILS
----------------------
Core operation:
```python
common_average = np.mean(data, axis=0, keepdims=True)  # Shape: (1, n_samples)
data_car = data - common_average  # Broadcasts to (n_channels, n_samples)
```

Effects:
- Reduces inter-channel correlations
- Removes reference electrode artifacts
- Preserves channel-specific (focal) signals
